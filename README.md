# 🔐 SecureVault

SecureVault is a Password Manager built using Python, Tkinter, and SQLite.

## ✨ Features

- Save Password
- Search Password
- Update Password
- Delete Password
- Show / Hide Password
- Generate Strong Password
- Copy Password to Clipboard

## 🛠️ Technologies Used

- Python
- Tkinter
- SQLite

## 🚀 How to Run

1. Install Python 3.
2. Download this project.
3. Open the project folder in VS Code.
4. Run:

```bash
python main.py
```

## 📷 Project

A simple desktop password manager with a user-friendly interface.
